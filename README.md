# online-course-manangement-system
Online Course Management System

📌 Overview

Online Course Management System is a web-based application developed during my BCA project work. It helps manage students, faculty, courses, and enrollments through a digital platform.

✨ Features

- Student Registration & Login
- Faculty Login
- Admin Dashboard
- Course Management
- Student Enrollment
- View Completed Courses
- File Uploads
- User Management

🛠️ Technologies Used

- PHP
- MySQL
- HTML
- CSS
- JavaScript


📂 Modules

- Admin
- User / Student
- Faculty
- Config
- Uploads

🚀 How to Run

1. Install XAMPP/WAMP
2. Place project folder in htdocs
3. Import database file in phpMyAdmin
4. Start Apache & MySQL
5. Open localhost in browser

- Full-stack web development basics
- Database connectivity
- Authentication system
- Project structuring
